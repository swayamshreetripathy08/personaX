# PERSONAX – Where Skincare Meets Personal Style

Hey there! 👋 Welcome to **PERSONAX**, a project I built to bridge the gap between inner confidence, personal style, and holistic self-care. 

I created this platform because I wanted to build something that goes beyond generic advice. By leveraging advanced data analysis of a user's unique attributes—from their skin profile and hair texture to their body styling and confidence levels—PERSONAX generates tailored fashion, skincare, and haircare recommendations.

I designed this project to deliver a smooth, luxurious, and highly personalized user experience, featuring glassmorphism elements, elegant glow animations, and responsive interactions.

---

## 🌟 What I Built (Features)

- **Multi-step Glow-Up Assessment**: I designed a dynamic multi-stage form that captures body type, skin profile, hair characteristics, style inspiration, and confidence metrics.
- **AI Glow Score**: A custom rating system I built to represent your initial personalization baseline.
- **Tailored Recommendations**: Real-time generation of custom styling, skincare, and haircare routines.
- **Modern UI/UX**: I focused heavily on aesthetics using soft pastel tones, luxury minimal design, smooth animations, and a responsive layout that works beautifully across all devices.
- **Glassmorphic Components**: Elegant frosted-glass cards that I crafted to highlight results and key features.

## 🛠 Technology Stack

Here are the tools and technologies I used to build this:

- **HTML5**: For semantic and accessible structure.
- **CSS3 / Vanilla CSS**: Custom variables, flexbox/grid layouts, custom animations (fade-in, slide-up, glow orbs), and responsive media queries.
- **JavaScript (ES6+)**: I wrote the logic for DOM manipulation, multi-step form validation, state management, and recommendation generation without relying on heavy frontend frameworks.
- **Google Fonts**: Inter & Outfit.
- **Boxicons**: For lightweight scalable icons.

# PERSONAX

**Where Skincare Meets Personal Style**

![PERSONAX Hero](https://via.placeholder.com/800x400.png?text=PERSONAX+Banner)

PERSONAX is a modern, premium, AI-powered personalization platform that bridges the gap between inner confidence and outer radiance. By leveraging advanced data analysis of your unique personal attributes—from skin profile and hair texture to body styling and confidence levels—PERSONAX generates tailored fashion, skincare, and haircare recommendations.

This project is built to deliver a smooth, luxurious, and highly personalized user experience, featuring glassmorphism elements, elegant glow animations, and responsive interactions.

---

## 🌟 Features

- **Multi-step Glow-Up Assessment**: A dynamic multi-stage form capturing your body type, skin profile, hair characteristics, style inspiration, and confidence metrics.
- **AI Glow Score**: A calculated rating representing your initial personalization baseline.
- **Tailored Recommendations**: Real-time generation of custom styling, skincare, and haircare routines.
- **Modern UI/UX**: Soft pastel tones, luxury minimal design, smooth animations, and a responsive layout designed for all devices.
- **Glassmorphic Components**: Elegant frosted-glass cards highlighting results and features.

## 🛠 Built With

- **HTML5**: Semantic and accessible structure.
- **CSS3 / Vanilla CSS**: Custom variables, flexbox/grid layouts, animations (fade-in, slide-up, glow orbs), and responsive media queries.
- **JavaScript (ES6+)**: DOM manipulation, multi-step form validation, state management, and recommendation logic without heavy frameworks.
- **Google Fonts**: Inter & Outfit.
- **Boxicons**: Lightweight scalable icons.

## 📥 Local Installation

To run this project locally, simply clone the repository and open the HTML file:

```bash
# Clone the repository
git clone https://github.com/yourusername/personax.git

# Navigate to the project directory
cd personax

# Open index.html in your preferred browser
# Alternatively, you can use Live Server extension in VS Code
```

## ☁️ Deployment on Render

This project is configured to be easily deployed on [Render.com](https://render.com/) utilizing Render's Static Sites.

1. Push this repository to your GitHub/GitLab account.
2. Log into Render, click **New +**, and select **Static Site**.
3. Connect your repository.
4. Render will use the provided `render.yaml` blueprint to immediately deploy your site.
5. (Optional) Custom domains can be configured in your Render dashboard.
